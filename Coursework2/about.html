About This Website

This website was created to educate people about Nintendo's game Animal Crossing: New Horizons, while also acting as an additional source of promotion for the Tropicana Animal Crossing Blog website. This website contains two main pages.

The Animal Crossng page which dives into the history of the game and how it came to be created, and the Tropicana page which is a blog styled page containing information about an island called Tropicana.

All information on this website was correct as of the time of being created. If any information is incorrect please contact us on our social media.
